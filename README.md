## Install

Clone or download the repository and then do:
```buildoutcfg
make install
```

If you want to install somewhere other than the defualt for your system:
```
make install prefix=install_prefix_directory
```

Required packages:
* `numpy`,
* `scipy`,
* `pandas`.
